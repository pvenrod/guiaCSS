Definición
CSS o "Cascading Style Sheets" es un lenguaje de hojas de estilo creado para controlar el aspecto de los documentos HTML. CSS es la mejor forma de separar los contenidos de su presentación y es imprescindible para crear páginas web complejas.

Un estilo CSS está formado de la siguiente manera:

selector {
     propiedad: valor;
}



Selectores

Los selectores son la referencia a la etiqueta, id o clase HTML a la que se le van a aplicar una serie de estilos CSS.

Existen 4 tipos de selectores básicos:
         - Selector universal: *
         - Selector basado en etiqueta: input   <input>
         - Selector basado en clase: .miClase   <input class="miClase">
         - Selector basado en ID: #miId   <input id="miId">

Estos selectores pueden ser agrupados para definir varias propiedades comunes a diferentes elementos.

Pseudoselectores

Los pseudoselectores funcionan de la misma forma que los selectores ya explicados, pero con una pequeña diferencia:
Hacen referencia a estados de un elemento que no pueden ser definidos en el código. Por ejemplo, si queremos que un enlace tenga un color diferente cuando ya ha sido visitadao, usaríamos el pseudoselector :visited.

Los pseudoselectores se escriben con dos puntos y detrás del selector, de la siguiente manera: selector:pseudoselector.
Selector Explicación Versión CSS de aparición

:link Un enlace que no ha sido visitado 1
:visited Un enlace que ya ha sido visitado 1
:fisrt-line La primera línea formateada de un elemento 1
:first-child El primer hijo de un elemento 2
:last-child El último hijo de un elemento 3
::before El contenido que hay antes de un elemento 2
::after El contenido que hay después de un elemento 2
:checked Un elemento marcado (como un checkbox) 2
