Definición
CSS o "Cascading Style Sheets" es un lenguaje de hojas de estilo creado para controlar el aspecto de los documentos HTML. CSS es la mejor forma de separar los contenidos de su presentación y es imprescindible para crear páginas web complejas.

Un estilo CSS está formado de la siguiente manera:

selector {
     propiedad: valor;
}



Selectores

Los selectores son la referencia a la etiqueta, id o clase HTML a la que se le van a aplicar una serie de estilos CSS.

Existen 4 tipos de selectores básicos:
         - Selector universal: *
         - Selector basado en etiqueta: input   <input>
         - Selector basado en clase: .miClase   <input class="miClase">
         - Selector basado en ID: #miId   <input id="miId">

Estos selectores pueden ser agrupados para definir varias propiedades comunes a diferentes elementos.
Selector Explicación Versión CSS de aparición
input Todos los elementos input 1
.miClase Todos los elementos con clase "miClase" 1
#identificador Todos los elementos con id "identificador" 1
input, strong Todos los elementos input y todos los elementos strong 1
#identificador, input Todos los elementos con id "identificador" y todos los elementos input 1
span strong Todos los elementos strong que desciendan de un elemento span 2
input.clave Todos los elementos input cuya clase sea "clave" 1
input[disabled] Todos los elementos input que contengan el atributo "disabled" 2
input[type="password"] Todos los elementos input que contengan el atributo "type" con el valor "password" 1
* Todos los elementos 2
div > span Todos los elementos span hijos de un elemento div 2
div + span Todos los elementos span precedidos de un elemento div 2


Pseudoselectores

Los pseudoselectores funcionan de la misma forma que los selectores ya explicados, pero con una pequeña diferencia:
Hacen referencia a estados de un elemento que no pueden ser definidos en el código. Por ejemplo, si queremos que un enlace tenga un color diferente cuando ya ha sido visitadao, usaríamos el pseudoselector :visited.

Los pseudoselectores se escriben con dos puntos y detrás del selector, de la siguiente manera: selector:pseudoselector.
Selector Explicación Versión CSS de aparición
:active Un elemento con el que está interactuando el usuario 1
:link Un enlace que no ha sido visitado 1
:visited Un enlace que ya ha sido visitado 1
:fisrt-line La primera línea formateada de un elemento 1
:first-child El primer hijo de un elemento 2
:last-child El último hijo de un elemento 3
::before El contenido que hay antes de un elemento 2
::after El contenido que hay después de un elemento 2
:checked Un elemento marcado (como un checkbox) 2
Propiedades

Propiedad position
Permite dar una poisición concreta a un elemento.
position
Valores aceptados: static | relative | absolute | fixed | sticky
Propiedades de tamaño
Permiten dar una altura y anchura concreta a un elemento.
width height
Unidades absolutas: cm | mm | in | px | pt | pc
Unidades relativas: em | ch | vw | vh | %
Propiedad font
Permite modificar el tipo de letra
font-family font-size font-weight font-style
CELIA VIÑAS

font-family: courier new;
font-size: 24px;
font-weight: 700;
font-style: italic;
Propiedades del texto
Permite modificar la apariencia del texto
color direction letter-spacing line-height
text-align text-decoration text-shadow line-height
CELIA VIÑAS

font-size: 30px;
font-weight: 700;
color: #ffffff;
text-shadow: 3px 5px 2px #grey;
Propiedad border
Permite modificar el estilo de los bordes de una caja
border border-color border-collapse border-radius
border-top border-bottom border-left border-right
border-style border-width border-spacing border-image
CELIA VIÑAS

border: 15px;
border-style: solid;
border-image: url('img/borde.png') 20% round;
CELIA VIÑAS

border: 5px dotted black
border-radius: 5px;
Márgenes interiores y exteriores
Permite modificar los márgenes de una caja
margin-top margin-bottom margin-left margin-right
padding-top padding-bottom padding-left padding-right
Propiedad background
Permite modificar el fondo de una caja
background-color background-image background-size background-repeat
